# Claw'Dance

```
 ▐▛███▜▌

```

Replaces Claude Code's "*Clauding..." spinner with a Claw'd dance animation.

## Installation

```bash
npm install -g clawdance
```

## Usage

```bash
# Install Claw'd dance
clawdance install

# Restore original spinner
clawdance uninstall

# Check installation status
clawdance status

# Preview animation
clawdance preview
```
